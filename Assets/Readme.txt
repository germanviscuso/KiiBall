KiiBall Demo
============

A demo of Kii Unity SDK for a web player based game. The game showcases
the following:

- Sign-in with Facebook
- Object storage and retrieval
- Analytics

For more info see Assets/Plugins/Readme.txt and:

http://blog.kii.com/?p=3576

Source code available at:

https://github.com/germanviscuso/KiiBall

Live game available at:

https://apps.facebook.com/kiiball
